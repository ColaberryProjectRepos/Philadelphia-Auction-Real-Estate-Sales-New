Line and Clustered Column Chart (Bedrooms) 22



Steps:


Copy and Paste Property Type Visual and make the appropriate changes

Build Visual Tab:


1. Set the x axis to "Bedrooms."
2. Assign the column y-AXIS values to "Sales Price."
3. Assign the line values to "Violent Crime Rate", Sale to Bid Ratio
4. Configure the tooltips to display "Sale to Bid Ratio", Violent Crime Rate, Rent Estimate, School Score
5. Rename the fields as necessary and remove any dot symbols.
Format Visual Tab: General:

Properties: Set the size to a height of 327 and width of 354.

Position: Horizontal position of 614 and vertical position of 391.

Title: Turn on the title and set the text to "Bedrooms" using the font "Segoe UI Bold" with a size of 14. The text color should be #FFFFFF (white), and the background color should be # #426871 Align the title in the center.
Visual:

X-axis: Values: Set the font to "Segoe UI" with a size of 9 and a font color of # 000000).

Turn off the title.

Y-axis: Values: Use the font "Segoe UI Bold" with a size of 9 and a font color of #000000 . Display units should be set to "Thousands".

Turn off the title.

Secondary Y-axis: Range: Enable the align zeros option.

Values: Use the font "Segoe UI Bold" with a size of 9 and a font color of #000000. Display units should be set to "Auto".

Turn off the title.

Legend: On

Gridlines: Turn off horizontal gridlines.

Columns: Set the colors to #426871, Turn On Show all

Lines: Set the colors for "Violent Crime Rate" to #D2B04C and "Sale to Bid Ratio" to #A3623A , Enable the shade area option.

Markers: Turn on markers.

Shape: Set the colors to #EB895F (Orange)

Data labels: Off
