Adding Logos and Title for the dashboard 15

Steps (Logos):

1. In Power BI Desktop, go to the Insert tab on the toolbar and click on Image
2. Choose the Colaberry logo from the provided zip file in the project essentials. Repeat the same process to select the Philly logo.
3. Open the Visualizations pane and select the General option. Fill in the following values:
Colaberry Logo: Size: Height=84, Width=162
Position: Horizontal=0, Vertical=0

Philly logo: Size: Height=84, Width=282
Position: Horizontal=997, Vertical=0


Steps (Title):

1. From Power BI Desktop toolbar under Insert, click on Text box from Elements
2. Type in ' Philadelphia Auction Real Estate ' in the text box
3. Select center alignment for the text (Font = Segoe UI Light, Font size = 32, Font Color = #252423
4. Open the Visualizations pane and select the General option. Fill in the following values:

Size: Height=76, Width=856
Position: Horizontal=162, Vertical=8
Effect: Background Color= #FFFFFF

5. Click on Format from Power Bi Desktop, Select Send Backward tab, and click on Send to back button
