Creating KPI, Sub KPI and Arrows 14




Steps:


1. Go to Report view in Power BI.
2. Right-click on the "Properties-Philly-Kraggle-v2" table in the Fields pane.
3. Select "New Measure" to create a new measure.
4. Use the provided DAX formulas to create the desired KPIs, sub-KPIs and Arrow and save them.
5. Repeat the above steps for each of the desired KPIs, sub-KPIs and Arrow.
DAX Formulas:
Sales Price = SUM(Properties_philly_Kraggle_v2[Average comps])/COUNT(Properties_philly_Kraggle_v2[Average comps])
Sale to Bid Ratio = DIVIDE(SUM(Properties_philly_Kraggle_v2[Average comps]),SUM(Properties_philly_Kraggle_v2[Sale Price/bid price]))
Rent Estimates = AVERAGE(Properties_philly_Kraggle_v2[Rent Estimate])
Violent Crime Rate = AVERAGE(Properties_philly_Kraggle_v2[ Violent Crime Rate ])
School Score = AVERAGE(Properties_philly_Kraggle_v2[ School Score ])
.PercentageChange SP =
VAR CurrentDate = MAX('Properties_philly_Kraggle_v2'[Sale Date])
VAR CurrentSalesPrice = [Sales Price]

VAR PreviousDate =
CALCULATE(
MAX('Properties_philly_Kraggle_v2'[Sale Date]),
FILTER(
ALL('Properties_philly_Kraggle_v2'),
'Properties_philly_Kraggle_v2'[Sale Date] < CurrentDate
)
)
VAR PreviousSalesPrice =
CALCULATE(
[Sales Price],
FILTER(
ALL('Properties_philly_Kraggle_v2'),
'Properties_philly_Kraggle_v2'[Sale Date] = PreviousDate
)
)
RETURN
IF(
NOT(ISBLANK(PreviousSalesPrice)),
DIVIDE(CurrentSalesPrice - PreviousSalesPrice, PreviousSalesPrice),
0
)
